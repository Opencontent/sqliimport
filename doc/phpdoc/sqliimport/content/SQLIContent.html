Proxy class for eZContentObject.

Useful for easy manipulation on content objects (insertion, modification...)  
  1.  // Create a new content in default language
    2. 

  2.  $contentOptions = new SQLIContentOptions( array( 'class_identifier' => 'article', 'remote_id' => 'my_remote_id' ) );
    3. 

  3.  $article = SQLIContent::create( $contentOptions );
    4. 

  4.  $article->fields->title = 'My article has a title';
    5. 

  5.  $article->fields->body = '<tag>myxmlcontent</tag>';
    6. 

  6.  
    7. 

  7.  // Create a new translation
    8. 

  8.  $article->addTranslation( 'fre-FR' );
    9. 

  9.  $article->fields['fre-FR']->title = 'Mon article a un titre';
    10. 

  10.  $article->fields['fre-FR']->body = '<tag>moncontenuxml</tag>';
    11. 

  11.  
    12. 

  12.  // Now create locations for content
    13. 

  13.  $article->addLocation( SQLILocation::fromNodeID( 2 ) ); // Will be main location
    14. 

  14.  $article->addLocation( SQLILocation::fromNodeID( 43 ) ); // Additional location
    15. 

  15.  
    16. 

  16.  // Now publish content
    17. 

  17.  $publisher = SQLIContentPublisher::getInstance();
    18. 

  18.  $publisher->publish( $article ); // Creator of this content will be currently logged in user
    19. 

To create XML for rich text attributes (XML Blocks), use SQLIXMLInputParser :  
  1.  // Let's assume that we have HTML content holded in $htmlContent variable
    2. 

  2.  $parser = new SQLIXMLInputParser();
    3. 

  3.  $parser->setParseLineBreaks( true );
    4. 

  4.  $document = $parser->process( $htmlContent ); // Result is a DOM Document
    5. 

  5.  $xmlContent = eZXMLTextType::domString( $document );
    6. 

  6.  
    7. 

  7.  // Now use it in your content
    8. 

  8.  $contentOptions = new SQLIContentOptions( array( 'class_identifier' => 'article', 'remote_id' => 'my_remote_id' ) );
    9. 

  9.  $article = SQLIContent::create( $contentOptions );
    10. 

  10.  $article->fields->title = 'My article has a title';
    11. 

  11.  $article->fields->body = $xmlContent;
